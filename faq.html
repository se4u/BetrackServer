General questions

How to install Betrack ?

  • 1) Please enable the Unknown sources setting on your phone. If you need more information please check the related section the FAQ.<
  • 2) Open a web browser on your Android device and navigate to http://www.ricphoto.fr/betrack/ and click on download.
  • 3) Once it's downloaded, open Downloads, tap on the APK file, and tap Yes when prompted. The app will begin installing on your device. If you don't manage to install the application that way you can use a file manager to install the application on your phone. If you need help on that topic please check the related section in the FAQ
  • 4) During the set up of Betrack you could be ask to enable the user stat usage, in that case please check the related section of the FAQ.
  • 5) If you see the screen below when installing Betrack, please click on Settings and follow the step describe in the section "How to enable the Unknown sources setting on your phone ?" of the FAQ:

Use a file manager to install BeTrack

  • 1) Usually all Android devices have a pre-installed File Manager app. Locate it in the app drawer, and tap it to open. If not you can use the Google Store to download a tool like ES file explorer.
  • 2) Locate the folder in which you have stored your APK file. It is probably located in the “Downloads” folder in your File Manager:
  • 3) Tap the APK file. A new screen will open showing the selected application’s name at the top. The screen will display more information about the app and permissions it needs to access.
  • 4) Tap the “Install” button on the bottom of the screen to install the application.
  • 5) If you see the screen below when installing Betrack, please click on Settings and follow the step describe in the section "How to enable the Unknown sources setting on your phone ?" of the FAQ:

How to allow the user stat usage ?

  • 1) During the first start of the application you might be asked to allow specific rights to Betrack:
    Tap on USAGE STAT
  • 2) You will be redirected to the page below:
    Tap on Betrack
  • 3) You will be redirected to the page below:
    Enable allow Usage access for Betrack
  • 5) Restart Betrack

How to white list betrack on a HUAWEI phone ?

  • 1) Durings the first start of the application you might be asked to white list Betrack:
    Select "do not show again" and tap on PROTECTED APPS
  • 2) You will be redirected to the page below:
    Select Betrack
  • 5) Restart Betrack

How to change the notification time ?

If by mistake you set up a notification that is not suitable for you please ask one of the research assistant. They can change that setting for you.

    Check the option Unknown sources Tap OK on the prompt message
